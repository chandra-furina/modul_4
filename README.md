# modul_4

4.1

Kode di atas adalah kode go yang menghitung dan menampilkan hasil dari permutasi dan kombinasi berdasarkan nilai input yang diberikan oleh pengguna. Fungsi faktorial digunakan untuk menghitung faktorial dari suatu bilangan, yang kemudian digunakan oleh fungsi permutasi dan kombinasi untuk menghitung nilai permutasi dan kombinasi.

4.2

Kode di atas adalah kode yang digunakan untuk menentukan pemenang dari kompetisi pemrograman berdasarkan jumlah soal yang berhasil diselesaikan dan total waktu pengerjaan. Peserta diberikan dalam bentuk string yang berisi nama dan waktu pengerjaan delapan soal. Fungsi "hitungSkor" adalah untuk menghitung jumlah soal yang diselesaikan "waktu pengerjaan di bawah 301 detik" serta total waktu pengerjaan soal-soal tersebut.

4.3

kode tersebut adalah kode yang mencetak deret bilangan berdasarkan aturan tertentu. Program meminta user untuk memasukkan sebuah bilangan bulat positif n, kemudian fungsi cetakDeret digunakan untuk mencetak setiap elemen dari deret. Jika n adalah bilangan genap, program membaginya dengan 2, dan jika ganjil, program mengalikannya dengan 3 lalu menambah 1. Proses ini berulang hingga nilai n mencapai 1.
